#ati-18n

Данное cli приложение создано для автоматического перевода JSON файлов. 
<br>
Приложение написано под пакет [i18n](https://www.npmjs.com/package/i18n). 
Оно позволяет при помощи разных переводчиков (на данный момент только Microsoft)
делать автоматический перевод JSON файлов с одного языка на другой. 

## Установка
npm
```sh
npm install ati-18n 
```
Так же пакет можно установить глобально, это позволит его использовать в любом проекте.

> Важно! После установки перезапустите терминал!

## Использование
> Если вам лень читать про методы которые есть, то вот краткая инструкция. 
> <br>
> Для перевода JSON файла, с русского на английский, при условии, что все файлы хранятся в корне папки. 
> <br>
> ati-18n --from ru --to en -r ./ru.json -w ./ 
> <br>
> Данная команда создаст en.json файл с готовым переводом. 
> ### Важно
> Программа на данный момент полностью перезаписывает существующий файл. 
> В следующих версиях я планирую исправить это и возможно сделать мягкую перезапись JSON файлов, но
> сейчас работайте аккуратнее.


####  translate

```sh
ati-18n translate 
```
Принимает опции

1. --from: с какого языка переводим, на данный момент названия указываются по ISO например ru, en и т.д
2. --to: на какой язык переводим, на данный момент названия указываются по ISO например ru, en и т.д
3. -r --read: с какого JSON файла переводим.
4. -w, --patch-write: путь в какую папку записывать файл (по умолчанию ./locales)

В целом необязательно будет каждый раз писать все опции.

#### generate-config
```sh
ati-18n generate-config 
```

Команда автоматически создает конфигурационный файл, если опции в методе translate не были указаны, 
то программа автоматически берет данные из сгенерированного файла.


```json
{
  "from": "en",
  "to": "ru",
  "read": "./locales/en.json",
  "patchWrite": "./locales"
}
```
В to можно так же указать массив, тогда программа сделает перевод на все языки сразу.  
```json
{
  "to": ["ru", "de"]
}
```
